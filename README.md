# Система учета заказов в 3D-студии

Графическое приложение на C++/Qt для учета клиентов, заказов, услуг и исполнителей в 3D-студии.

Проект разработан в рамках лабораторной работы №8 по дисциплине «Объектно-ориентированное программирование». Основная цель — показать работу с объектами предметной области, связями между ними и паттернами проектирования.

## Возможности

Приложение позволяет:

- добавлять, редактировать, удалять и искать клиентов;
- добавлять, редактировать, удалять и искать заказы;
- добавлять, редактировать, удалять и искать услуги;
- добавлять, редактировать, удалять и искать исполнителей;
- добавлять услуги в выбранный заказ;
- назначать исполнителей на заказ;
- просматривать заказы выбранного клиента;
- просматривать заказы по выбранной услуге;
- просматривать заказы выбранного исполнителя;
- рассчитывать стоимость заказа разными способами.

## Основные сущности

В проекте используются следующие классы предметной области:

- `Client` — клиент студии;
- `Order` — заказ клиента;
- `Service` — услуга студии;
- `Employee` — исполнитель;
- `OrderService` — связь между заказом и услугой;
- `Assignment` — связь между заказом и исполнителем.

В проекте реализованы связи «многие ко многим»:

- заказ может содержать несколько услуг, а услуга может входить в разные заказы;
- заказ может выполняться несколькими исполнителями, а исполнитель может участвовать в разных заказах.

## Паттерны проектирования

В проекте применены три паттерна:

- `Factory Method` — создание диалоговых окон через `DialogFactory`;
- `Facade` — упрощенный доступ к операциям через `StudioFacade`;
- `Strategy` — разные способы расчета стоимости заказа через `CostCalculationStrategy`.

## Сборка

Открыть проект можно через Qt Creator:

1. Открыть файл `application/CMakeLists.txt`.
2. Выбрать комплект сборки Qt.
3. Собрать и запустить проект.

Также можно собрать через терминал:

cd application
mkdir build
cd build
